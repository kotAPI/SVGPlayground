Inserting in a BST

Psuedo-code:
                  insert (node, root){
                      if root == null
                          root = node
                      else if node.data == root.data
                          Print "Already Exists!"
                      else if node.data < root.data
                          insert(node, root.left)
                      else if node.data > root.data
                          insert(node, root.right)
                  }

Searching in a BST

Psuedo-code
                search (key, root){
                  if root == null
                      Print "Not found!"
                  else if key == root.data
                      Print "Found!"
                  else if key < root.data
                      search(key, root.left)
                  else if key > root.data
                      search(key, root.right)
               }
