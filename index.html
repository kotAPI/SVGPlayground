Insert And Searching Linked List

Inserting at the end of a Linked List

Psuedo-code:
                insert (int data){
                if head == null
                  head = new Node(data)
                else 
                  temp = head
                  while(temp.next != null)
                    temp = temp.next
                  element = new Node(data)
                  temp.next = element
                }

Searching in a Linked List

Psuedo-code:
                  search(int data){
                    temp = head;
                    while(temp != null)
                        if(temp.data == data)
                            Print "Found!"
                        else
                            temp = temp.next
                    
                    if(temp == null)
                        Print "Not Found!"
                }
